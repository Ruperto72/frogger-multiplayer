Frog vs Toad — Music Editor

Help

Dev tool for composing the "Froggy Hop" loop. Loads the current song from js/audio.js. All four tracks are shown at once in the piano roll — the checkboxes in the toolbar control both visibility and which tracks are audible when playing, while "Active track" controls which track the tools below draw/erase/move in (other tracks are shown dimmed and can't be clicked).

Three tools: Pen adds a new note/hit in an empty cell, Eraser removes a note/hit you click on (or a whole marquee-selected group, see below), Grab moves a note/hit by dragging it (drag its right edge instead to change its length). In Grab, dragging on empty space marquee-selects every note/hit in the active track whose start falls within the dragged time range — the group then moves together, and Ctrl/Cmd+C / Ctrl/Cmd+V copies and pastes it (anchored at the playhead). Select a single note (with Pen or Grab) to edit vibrato/tremolo/bend/duty cycle/arpeggio/portamento/bitcrush/echo/ chorus in the panel on the right. Portamento glides unbroken into the very next note in the same track (no gap between them) instead of retriggering; it can't be combined with bend or arpeggio on the same note. The rhythm strip has four rows (Kick/Snare/Hi-hat/Tom).

Keyboard shortcuts (ignored while typing in a field): Space play/stop, 1/2/3 select Pen/Eraser/Grab, Delete/Backspace delete the selection, arrow keys nudge it by one column/semitone, Ctrl/Cmd+C/V copy/paste, Escape deselect (or close this dialog).

The red line above the grids is a playhead — drag it, or click in the ruler row, to choose where playback starts; it moves along with the music during playback. The green band in the ruler is the loop range — drag its ends to loop just a section, or "Full loop range" to reset it to the whole song. The 🔍 buttons zoom the piano roll horizontally.

"Save file" downloads what you're working on as a .json file, "Load file" reads such a file back in — separate from "Export code", which instead generates finished code to paste into LEAD/HARMONY/BASS/RHYTHM in audio.js. Your work is also autosaved to this browser continuously as a safety net against an accidental reload.
